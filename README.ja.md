# METI-JPO オープンイノベーション向けモデル契約書

本リポジトリは、経済産業省（METI）および特許庁（JPO）が策定した「研究開発型スタートアップと事業会社のオープンイノベーション促進のためのモデル契約書」の改訂ドラフト版をホストしています。本リポジトリは、契約書改善のためのフィードバックを収集するべく、GitHubを通じた公開意見募集の試行に利用されました。

## 意見募集（アーカイブ済み）

本プロジェクトの公式な意見募集期間は、**2023年5月17日から2023年6月18日**まででした。本リポジトリは現在、公開参考資料としてアーカイブされています。

当初の目的は、契約条項の内容やその背景、特に交渉において争点となりやすい条項に関するフィードバックを収集することでした。

### 投稿方法（当時のプロセス）

意見の投稿は、変更案を含む[pull request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request)を作成する形で行われました。

- **[詳細な投稿手順については MANUAL.md をご覧ください](MANUAL.md)**
- 投稿前にすべての参加者に対し、特定の事例や当事者が特定されないよう、[運用方針](OPERATION_POLICY.md)を確認することが求められました。

## 対象読者

本プロジェクトは、特に以下の方々を対象としています：
- 研究開発型スタートアップおよび事業会社の知財・法務担当者
- 弁護士やベンチャーキャピタルなど、スタートアップエコシステムの関係者
- 本モデル契約書を利用したことがある、または関心のある方

## 本プロジェクトについて

2020年度（令和2年度）、経済産業省および特許庁は、研究開発型スタートアップと事業会社間の協業交渉における参考資料として、オリジナルの[モデル契約書（新素材編・AI編）](https://www.jpo.go.jp/support/general/open-innovation-portal/index.html)を策定しました。

本プロジェクトは、契約書のさらなる改善に向けて、現在および将来のユーザーから幅広いフィードバックを収集するために立ち上げられました。GitHubを活用することで、コミュニティとのオープンかつ継続的な対話を生み出し、より実践的でニーズに即した契約書のアップデートを可能にすることを目指しました。

この取り組みは、オープンソースの原則を活用し、ビジネス環境の変化やイノベーターのニーズに柔軟に対応する、現代的な政策立案のアプローチを示すものです。

![モデル契約書改訂プロセス](https://user-images.githubusercontent.com/84115514/118421248-bedaf800-b6fb-11eb-846f-580a8eee51f5.png)

*本試行を通じて寄せられた意見は委員会で検討され、モデル契約書改訂のための参考資料として活用されました。*
